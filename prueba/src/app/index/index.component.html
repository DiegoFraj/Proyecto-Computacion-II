Bienvenido a mi aplicación


¿De qué trata el análisis de sentimientos?

El análisis de sentimiento implica la construcción de un sistema para recoger y categorizar opiniones sobre un producto. Conocido también como minería de opinión, puede automatizarse usando machine learning, un tipo de inteligencia artificial (IA), para extraer el sentimiento del texto.

Aplicaciones

Con este sistema se podría identificar el éxito o fracaso de un proyecto dependiendo de la opinión pública que, a través de encuestas o comentarios por la web, el cliente aporta.
